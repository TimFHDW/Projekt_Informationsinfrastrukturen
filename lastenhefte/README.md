# lastenhefte/ – Quellmaterial

Die Lastenhefte sind die **einzige inhaltliche Referenz** für Generierung und Evaluation.

## Regeln

- **Nur lesen, niemals ändern.** Auch keine Formatkorrekturen.
- Beim Prompten wird das jeweilige Lastenheft vollständig an der Stelle `{LASTENHEFT}`
  eingesetzt bzw. angehängt (siehe `prompts/README.md`).
- Die inhaltliche Korrektheit (K2) und Vollständigkeit (K3) der Diagramme wird immer
  **gegen das Lastenheft** geprüft, nie gegen eigenes Wissen über das System.
