# evaluation/ – Kriterien, Matrix, Quervergleiche

## Inhalt

| Datei/Ordner | Zweck |
|---|---|
| `kriterien.md` | Operationalisierte Bewertungskriterien K1–K4 mit Skalen – **Maßstab für alle Evaluationen** |
| `ergebnismatrix.md` | Statusübersicht aller 36 Zellen inkl. Scores – nach jeder Änderung pflegen |
| `vergleiche/` | Auswertungen ÜBER die Einzelbewertungen hinweg |

## Regeln

- Einzelbewertungen liegen **nicht** hier, sondern als `evaluation.md` in der jeweiligen
  Zelle unter `ergebnisse/`.
- Die Quervergleiche werden erst geschrieben, wenn genügend Einzelbewertungen vorliegen
  (Auswertungsphase, ca. ab 16.07.). Sie stützen sich ausschließlich auf die
  `evaluation.md`-Dateien und die Matrix – keine neuen Behauptungen ohne Beleg in den Zellen.
- Jede Aussage in einem Vergleich soll auf konkrete Zellen verweisen
  (z. B. `ergebnisse/easyride/claude/sequenz/`).
